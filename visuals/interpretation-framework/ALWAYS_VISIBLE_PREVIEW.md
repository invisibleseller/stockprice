# 永远可见版：解读与情绪课程框架

这个版本不用 SVG、不用 HTML、不用图片渲染。只要能看到文字，就能看到课程图的结构。

## 最核心主图

```text
┌──────────────────────────────────────────────┐
│              看见“解读”                       │
│          情绪就有了选择空间                    │
└──────────────────────────────────────────────┘

        ① 事实发生
        例：婆婆说“天气好了，你出去玩玩吧”
              ↓
        ② 自动解读
        “她是不是在嫌弃我？”
              ↓
        ③ 情绪产生
        委屈 / 生气 / 防御 / 想反击
              ↓
        ④ 被情绪带走
        冷淡 / 反击 / 生闷气 / 内耗
              ↓
┌──────────────────────────────────────────────┐
│  暂停一下：这是事实，还是我的一种解读？        │
└──────────────────────────────────────────────┘
              ↓
        ⑤ 看见：这只是我的一种解读
              ↓
        ⑥ 尝试其他可能
              ↓
┌──────────────┬──────────────┬──────────────┐
│  可能 A       │  可能 B       │  可能 C       │
│  她在关心我   │  她想让我放松 │  她表达不好   │
│  → 温暖       │  → 轻松一点   │  → 没那么受伤 │
└──────────────┴──────────────┴──────────────┘
              ↓
        ⑦ 情绪松动
              ↓
        ⑧ 我有新的回应选择
```

## 一句话例子图

```text
事实：婆婆说“过两天天气好了，你出去玩玩吧”
              ↓
       同一句话，可以有不同解读
              ↓
┌──────────────┬──────────────┬──────────────┐
│ 解读 1        │ 解读 2        │ 解读 3        │
│ 她关心我      │ 她嫌弃我      │ 她讽刺我爱玩  │
│              │              │              │
│ 情绪：温暖    │ 情绪：委屈愤怒│ 情绪：羞耻防御│
│ 反应：回应    │ 反应：冷淡    │ 反应：反击    │
└──────────────┴──────────────┴──────────────┘
```

## 三层模型图

```text
┌──────────────────────────────────────────────┐
│ 表层：情绪                                    │
│ 愤怒 / 委屈 / 失落 / 防御                     │
└──────────────────────────────────────────────┘
              ↓
┌──────────────────────────────────────────────┐
│ 中层：解读                                    │
│ 不尊重我 / 不在乎这个家 / 我孩子不行 / 故意的 │
└──────────────────────────────────────────────┘
              ↓
┌──────────────────────────────────────────────┐
│ 底层：事实                                    │
│ 孩子没回应 / 伴侣晚归 / 老师反馈 / 孩子哭闹   │
└──────────────────────────────────────────────┘
```

## 两条路径对照图

```text
同一件事发生
       ↓
┌──────────────────────┬──────────────────────┐
│ 看不见解读            │ 看见解读              │
├──────────────────────┼──────────────────────┤
│ 事实                  │ 事实                  │
│ ↓                     │ ↓                     │
│ 负面解读              │ 发现：这是解读        │
│ ↓                     │ ↓                     │
│ 负面情绪              │ 寻找其他可能          │
│ ↓                     │ ↓                     │
│ 自动反应              │ 情绪松动              │
│ ↓                     │ ↓                     │
│ 关系受伤 / 自我内耗   │ 有选择地回应          │
└──────────────────────┴──────────────────────┘
```
